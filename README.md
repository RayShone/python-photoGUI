# python-photoGUI

图片转字符串程序 —— 将图片的每个像素点替换为自定义字符，生成字符画风格的图片。

## 功能特性

- 🖼️ 支持常见图片格式（JPG / PNG / BMP / GIF / TIFF / WebP）
- ✏️ 自定义替换字符串（支持中文、英文、符号等任意字符）
- 🔤 中/英文模式一键切换（内置默认字符表）
- ⚙️ 可调字体大小和采样步长（步长越大速度越快，步长越小越精细）
- 🧵 后台线程转换，转换过程 GUI 不冻结
- 🖥️ 纯 Python + tkinter 实现，跨平台兼容

## 依赖

```bash
pip install pillow numpy
```

## 运行

```bash
python photoGUI.py
```

## 使用说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 自定义字符串 | 用于替换像素的字符，留空使用默认字符表 | 空（使用模式默认） |
| 字体大小 | 字符渲染字号 | 10 |
| 采样步长 | 每隔 N 个像素采样一次，越大越快 | 3 |
| 中/英文模式 | 自定义字符串为空时，使用的默认字符表 | 中文模式 |

1. 启动程序后，输入自定义字符串（可选）
2. 选择中/英文模式（影响默认字符表）
3. 调整字体大小和采样步长（可选）
4. 点击「选择图片并开始转换」
5. 等待转换完成，自动显示结果并保存为 `converted_image.jpg`
